Q1: Why might we consider using Git?

Answer: Data sciences would consider using Git because it provides them with a 
        way of keeping track of files for multiple people. Additionally it 
        allows for files to be recovered if there is corrupt data.

Q2: What field could data science be applied to? Give an Example.

Answer: Data Science could be applied to the health care field because in the 
        health care field there is a big need to obtain a lot of data about 
        people and their health conditions, to then be able to conduct research,
        which is where data science can be applied.

Q3: What is the difference between Merging and Rebasing?

Answer: Merging takes two branches and attempts to combines the changes from 
        both branches. Rebasing takes one branch's whole version history and put
        it on another branch.
